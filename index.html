Day Scheduler

Full-Stack Web Development

This is a web application created in the Northwestern University Coding Program.

This day scheduler will react dynamically with you depending on what time of day it is! Never wonder, "Is it 1pm? Is it 2pm?", simply glance at your calender and let the red color signal you in the right direction. Want to save some info for 4pm? Go ahead and type in whatever you want and hit that save button! As long as you are on the same device, the schedule will remember what you have saved.
